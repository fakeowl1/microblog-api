# Functional вимоги до системи

## 1. Управлінням профілем користувача 

- Реєстрація нового користувача
- Автентифікація користувача
- Перегляд профілю та редагування власного профілю
- Видалення облікового запису

## 2. Управління контентом (пости) 

- Створення нового поста
- Перегляд певного посту
- Перегляд власної стрічки
- Редагування власного поста
- Видалення власного поста

## 3. Управління соціальними зв'язками

- Надіслати запит на дружбу
- Прийняття/відхилення запиту на дружбу
- Видалити користувача зі списку друзів
- Перегляд списку друзів та їхніх профілів

## 4. Взаємодія з контентом

- Можливість лайкнути/дизлайкнути пост
- Можливість залишити коментар під постом
- Можливість лайкнути/дизлайкнути комментар
- Можливість відповісти на комментар

## 5. Системні функції

- Пошук користувача за іменем
- Сповіщення про нові лайки, дизлайки, комментарі, запити в друзі, тощо
