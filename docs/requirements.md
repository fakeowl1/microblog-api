# Functional вимоги до системи

## 1. Управлінням профілем користувача 

- FR1: Реєстрація нового користувача
- FR2: Автентифікація користувача
- FR3: Перегляд профілю та редагування власного профілю
- FR4: Видалення облікового запису

## 2. Управління контентом (пости) 

- FR5: Створення нового поста
- FR6: Перегляд певного посту
- FR7: Перегляд власної стрічки
- FR8: Редагування власного поста
- FR9: Видалення власного поста

## 3. Управління соціальними зв'язками

- FR10: Надіслати запит на дружбу
- FR11: Прийняття/відхилення запиту на дружбу
- FR12: Видалити користувача зі списку друзів
- FR13: Перегляд списку друзів та їхніх профілів

## 4. Взаємодія з контентом

- FR14: Можливість лайкнути/дизлайкнути пост
- FR15: Можливість залишити комментар під постом
- FR16: Можливість лайкнути/дизлайкнути комментар
- FR17: Можливість відповісти на комментар

## 5. Системні функції

- FR18: Пошук користувача за іменем
- FR19: Сповіщення про нові лайки, дизлайки, комментарі, запити в друзі, тощо
